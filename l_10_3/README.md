# Задание 10.3 — Полный пайплайн: clustering_hw.csv

**Запуск:** из папки `L_10_3` выполните:
```bash
python main_10_3_full_pipeline.py
```

**Что делает скрипт:**
1. Загружает `data/clustering.csv` и строит график данных (сохраняется в `figures/clustering_data.png`)
2. Подбирает оптимальное k по метрике силуэтта (перебор k от 2 до 10)
3. Обучает K-means с выбранным k и сохраняет визуализацию в `figures/clustering_kmeans.png`
4. Определяет кластеры для точек (5, 8) и (0, 5) и выводит, один кластер или разные
5. Считает качество по AMI, если в CSV есть колонка с истинными метками

Графики сохраняются в папке `L_10_3/figures/`.
