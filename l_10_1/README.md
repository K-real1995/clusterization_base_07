# Метрика силуэтта (Silhouette)

**Запуск:** из папки `L_10_1` выполните:
```bash
python main_10_1_silhouette.py
```

**Что делает скрипт:**
- Загружает `data/clustering.pkl`
- Обучает K-means при k=2 и k=4
- Считает метрику силуэтта для обоих вариантов
- Выводит, при каком k метрика лучше (по силуэтту лучше то k, при котором значение выше)

**Итог по вашему датасету:** при k=2 силуэт выше, чем при k=4 — по этой метрике лучше два кластера.
