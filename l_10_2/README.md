# L_10_2 — Метрика AMI (Adjusted Mutual Information)

**Запуск:** из папки `L_10_2` выполните:
```bash
python main_10_2_ami.py
```

**Что делает скрипт:**
- Загружает `data/clustering.pkl` (если в файле есть ключ `y` или `labels` — использует их как истинные метки)
- Обучает K-means при k=2 и k=4
- Считает AMI между предсказанными и истинными метками
- Выводит, при каком k метрика AMI лучше

**Важно:** AMI имеет смысл только при наличии эталонных меток в данных. Если в `clustering.pkl` нет ключа с метками, скрипт сообщит об этом и сравнение по AMI вывести нельзя.
